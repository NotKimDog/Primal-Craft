## üìñ How to Obtain Enchantments

### Enchanting Table
1. Place enchanting table
2. Place item in slot
3. Select enchantment
4. Pay XP cost
5. Enchantment applied

**Max Enchantments:** 3 per item (higher levels get fewer options)

### Anvil Combining
1. Place anvil
2. Put item + enchanted book in slots
3. Select result
4. Pay XP cost
5. Combined item created

**Costs:**
- First enchantment: Base cost
- Each additional: +5 XP
- Total: Up to 40 XP for complex combinations

### Enchanted Books
1. Find in loot chests
2. Fish for books
3. Trade with librarians
4. Apply to items with anvil

### Loot Chests
- Dungeons: Common enchanted items
- Temples: Rare enchantments
- End City: Top-tier enchantments
- Nether Fortress: Fire-related enchants

## üîß Enchantment Conflicts

### Cannot Combine
- **Silktouch + Fortune** - Mining alternatives
- **Fire Aspect + Frost Walker** - Temperature conflict
- **Mending + Infinity** - Repair mechanics
- **Multiple Protection Types** - Limited to 4 levels total

### Stacking Rules
- **Protection:** All types stack (max 4 each)
- **Damage Enchants:** Don't stack
- **Tool Enchants:** Most stack with others
- **Armor Enchants:** All can combine
